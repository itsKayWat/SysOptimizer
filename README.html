SysOptimizer

Overview

SysOptimizer is a comprehensive Windows optimization tool designed to enhance your computer's performance. It cleans temporary files, adjusts system settings for optimal performance, runs disk cleanup, and provides detailed system health reports‚Äîall with a single script.

Why SysOptimizer?

Over time, Windows systems can become sluggish due to accumulated temporary files, suboptimal settings, and fragmented disks. SysOptimizer automates the cleanup and optimization processes, ensuring your PC runs smoothly and efficiently without manual intervention.

Features

  • Clear Temporary Files: Removes unwanted temporary files from multiple system directories.
  • Optimize Performance Settings: Disables unnecessary visual effects and sets the power plan to high performance.
  • Run Disk Cleanup: Executes Windows Disk Cleanup utility for thorough maintenance.
  • System Health Check: Provides real-time metrics on CPU, memory, and disk usage.
  • User-Friendly: Simple command-line interface with clear prompts and options.

Installation

Requirements

  • Operating System: Windows 10 or later
  • Python: Version 3.6 or higher

Steps

git clone https://github.com/yourusername/SysOptimizer.git
cd SysOptimizer
pip install -r requirements.txt

Usage

  1. Run the Script as Administrator
    • Right-click on SysOptimizer.py and select Run as administrator.
    • OR via Command Prompt: 
      python SysOptimizer.py
  2. Follow On-Screen Prompts
    • The script will perform the following:
      • Clear temporary files.
      • Optimize performance settings.
      • Run Disk Cleanup.
      • Display system health before and after optimization.
    • Optionally, restart your computer to apply all changes.

Example Scenario

Customer Situation: Jane notices her Windows PC has become slow over time, especially when opening applications and browsing files. She wants a quick solution to clean up her system and boost performance without manually tweaking settings.
